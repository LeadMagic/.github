# LeadMagic `.github` meta repository

This repository only exists to host the LeadMagic organization profile and shared GitHub configuration. It is **not** a product repository.

- **Rendered profile README** — [`profile/README.md`](profile/README.md) is what GitHub displays on the organization page at [github.com/LeadMagic](https://github.com/LeadMagic).
- **Shared org configuration** — if you're looking for community health files (issue templates, funding info, support links) for the LeadMagic org, this is the right place to add them. See [GitHub's docs on the `.github` repository](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

## Contributing to the profile

Edits to the public org page go through `profile/README.md`. Please keep copy aligned with the current product positioning on [leadmagic.io](https://leadmagic.io) and the brand-impersonation allow-list in [`leadmagic-openapi/SECURITY.md`](https://github.com/LeadMagic/leadmagic-openapi/blob/main/SECURITY.md).
